VR Design Room

By Reality Check

A virtual reality room design application where the user is able to query 3D models using a voice interface.




Overview


  • Query and manipulate multiple objects to form beautiful and detailed environments such as office designs, home interior decor, or machine workshops
  • Designed for Oculus Rift and HTC Vive

Controls



Start Room


  • Press "Right Trigger" to interact with objects including picking up objects in the scene.
  • Interact with the door knobs to create a new Box, Demo, or Outside room.
  • Interact with one of the three Tvs in the room to load up a previously saved rooms.


New Rooms


  • Press "Right Trigger" to interact with objects.
    • After selecting an object in the scene using the "Right Trigger", a properties panel will appear. From here the position, rotation, and scale of the selected model can be finely tuned. The physics of the model can be adjusted here. Toggling the "Obey Gravity" check box will control whether or not the model can be affected by outside forces such as gravity and other scene models. Here the object can be deleted as well.
  • Press "Right Trigger" or "Left Trigger" to pickup objects in the scene.
  • To bring up the menu. press "Y" or "B" for Oculus Rift, or either of the top menu buttons for HTC Vive.
    • From the menu, you can either go back to the start up lobby, change the volume of the music, or add a new model to the scene.
  • Search Menu
    • Interact with the text field to bring up the virtual keyboard
    • Interact with the mic button to start the voice recording. This records for 5 seconds and then transcribes the voice recording into text. The recording can be manually halted by simply pressing the mic button again.
    • After the desired search text has been entered using the keyboard, press the search icon to initiate the search algorithm, and simply interact with one of the search results to load the respective model into the scene.
    • If there are more than 4 search results, interact with the next page and last page buttons to view more search results.
  • Tutorial Menu
    • Interact with one of the buttons to view the respective tutorial video.
    • Press "Y" or "B" for Oculus Rift, or either of the top menu buttons for HTC Vive to cancel the video.
  • Save Menu
    • Interact with the text field to bring up the virtual keyboard and type the desired name of the save.
    • Press the save button to save the current state of the room.
    • If there are already 3 save slots, select one of the slots to override.
  • To close the menu, press "Y" or "B" for Oculus Rift, or either of the top menu buttons for HTC Vive, or Interact with the close button in the top right corner of each of the panels to close the active panel.
